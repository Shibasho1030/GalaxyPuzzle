## ゲーム概要

`GalaxyPuzzle` は、落下してくるブロックを操作してラインを揃え、スコアを伸ばしていくパズルゲームです。

通常のテトリス風ルールに加えて、爆弾ブロックやハードモードなどの独自要素を追加し、単なる模写ではなく、自分なりにゲーム性を拡張した作品です。

## 主な機能

### 通常モード

基本的なテトリス風のルールで遊べるモードです。

- ブロックの左右移動
- ブロックの回転
- ブロックの落下
- ライン消去
- スコア加算
- 落下速度の上昇
- ゲームオーバー判定
- ゲームリスタート

## 自力で実装した機能

### ハードモード

通常のテトリスブロックだけではなく、さまざまな大きさ・形のブロックで遊べるモードです。

通常モードよりも配置判断が難しくなるため、よりパズル性の高いゲーム体験を目指しました。

### 爆弾機能

通常のライン消去とは別に、爆弾ブロックによる特殊な消去機能を実装しました。

単純にブロックを積むだけでなく、爆弾をどのように使うかを考える戦略性を追加しています。

### 次のブロック表示

次に出現するブロックだけでなく、2つ目以降のブロックも表示する機能を実装しました。

プレイヤーが先を読んで配置を考えられるようにし、ゲームの戦略性を高めています。

### 落下位置の自動表示

現在操作しているブロックが、どの位置に落下するかを自動で表示する機能を実装しました。

これにより、プレイヤーが配置位置を判断しやすくなり、操作ミスを減らせるようにしました。

### キー設定保存機能

プレイヤーが設定したキー操作を保存できるようにしました。

毎回キー設定を変更しなくてもよいようにし、継続して遊びやすい体験を目指しました。

### BGM・効果音

ゲーム中のBGMや効果音を実装しました。

ブロック操作、消去、ゲーム進行などに音を加えることで、ゲームとしての没入感を高めています。

### 背景画像・世界観の作成

背景画像やBGMを組み合わせ、単なるテトリスではなく、宇宙・ギャラクシー感のあるゲーム世界観を意識して実装しました。

## 工夫した点

### 1. 壁際で回転したときのバリデーション

ブロックを壁際で回転させたときに、盤面外にはみ出したり、既存ブロックと重なったりしないように判定処理を作成しました。

ただ回転させるだけではなく、

- 回転後の座標が盤面内に収まっているか
- 他のブロックと重なっていないか
- 回転できない場合は処理をキャンセルする

といったバリデーションを考察しました。

この部分は、テトリス系ゲームにおいて操作感に直結する重要な処理だと考え、特に意識して実装しました。

### 2. 爆弾機能の設計・実装

通常のライン消去だけではゲーム性が単調になりやすいと考え、爆弾機能を追加しました。

爆弾によって周囲のブロックを消せるようにすることで、プレイヤーが危機回避や高得点を狙えるようにしています。

### 3. ハードモードの作成

通常のテトリスブロックだけではなく、さまざまな大きさ・形のブロックで遊べるハードモードを作成しました。

これにより、同じ盤面でも通常モードとは違った判断が必要になり、難易度とリプレイ性を高めました。

### 4. 操作感の改善

左右移動時に、キー入力を受け取るたびに連続で移動しすぎないように調整しました。

一度横移動したあと、一定時間は横移動しないようにすることで、意図しない連続移動を防ぎ、操作感を改善しています。

### 5. スコア設計

ライン消去やゲーム進行に応じてスコアが加算される仕組みを設計・実装しました。

ただ消すだけではなく、プレイヤーが高得点を目指したくなるようにゲーム性を加えています。

### 6. 落下速度上昇機能

ゲームが進むにつれてブロックの落下速度が上がるようにしました。

時間経過や進行状況によって難易度が上がることで、ゲーム後半の緊張感を作っています。

### 7. ゲームリスタート機能

ゲームオーバー後にすぐ再挑戦できるよう、リスタート機能を実装しました。

プレイヤーがテンポよく遊び直せるようにすることで、ゲーム体験の改善を意識しました。
